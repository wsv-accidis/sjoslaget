Tredje gången gillt...

...men inte för vare sig HK eller DP; efter att vi har fått ställa in såväl Vårkryssen som AG håller vi nu tummarna för tredje gången gillt, att mörker byts mot ljus (samtidigt som ljus byts mot mörker) och att vi den 11:e oktober kan få genomföra det bästa Sjöslaget någonsin!

Förra årets förändring lär inte ha undgått någon. Det var fler artister, fler DJ:s och mer som hände i största allmänhet. I år fortsätter vi det spåret och ser till att det blir ett fullspäckat dygn ombord med artister, DJ:s, stand up, pop-up-klubb, speed dating, tema, filmvisning, pub quiz och mycket mer.

Alla löpande uppdateringar hittar du i vårt Facebook-event. I sinom tid så läggs dessa uppdateringar ihop till ett fullständigt program som du hittar här på sidan.
