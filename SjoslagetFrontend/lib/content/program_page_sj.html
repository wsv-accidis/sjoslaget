Program

Vi gör allt för att de dygnet ombord ska bli så späckat av aktiviteter som möjligt. Löpande uppdateringar hittar du i vårt Facebook-event. I sinom tid läggs dessa ihop till ett fullständigt program som du hittar här nedan. Håll till godo med ett preliminärt program så länge. Du som varit med innan kan förvänta dig ungefär samma hålltider som tidigare.

Kryssningsprogram

  • 16:00 Biljettutlämning startar i terminalen.
  • 19:00 Ombordstigning (det händer att den fördröjs lite).
  • 19:30 Fartyget avgår!
  • 19:30 - 18:00 Vi har kul!
  • 18:00 Fartyget anländer till Stockholm.

För er som bokar hemtransporter, var medvetna om att tiden då vi anländer är preliminär. Den brukar stämma, men det tar också tid att komma av fartyget så ta med det i era beräkningar.
