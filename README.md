Sjöslaget
=========

_This application is directed exclusively at a Swedish audience. International visitors, please
contact me if you have any questions._

Det här skall bli den officiella webbplatsen för [Sjöslaget](http://www.sjoslaget.se). Sjöslaget 
är en årlig kryssning för i huvudsak universitets- och högskolestudenter.

## Status
Sidan är under utveckling och inte i drift ännu. Sidan på [www.sjoslaget.se](http://www.sjoslaget.se)
är föregående version som inte delar någon kod med denna.
